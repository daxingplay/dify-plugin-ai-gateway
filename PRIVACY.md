# Privacy Policy

## Overview

This privacy policy describes how the Dify Plugin for Alibaba Cloud AI Gateway ("the Plugin") handles data when you use it within the Dify platform.

## Data Collection by the Plugin

**The Plugin itself does not collect, store, or retain any personal information or user data.** 

The Plugin serves solely as a gateway/proxy that facilitates communication between your Dify instance and Alibaba Cloud AI services.

## Data Transmission to Alibaba Cloud Services

When you use this Plugin, your data is transmitted to Alibaba Cloud for processing. The types of data that may be transmitted include:

- **Text prompts and responses** (when using LLM services)
- **Text content** (when using text embedding or rerank services)
- **Audio files** (when using speech-to-text or text-to-speech services)
- **API credentials** (authentication tokens stored in your Dify configuration)
- **Model parameters and configuration settings**

All data transmission occurs directly between your Dify instance and Alibaba Cloud's servers. The Plugin does not intercept, store, or log this data.

## Alibaba Cloud Privacy Policies

Your data transmitted through this Plugin will be processed by Alibaba Cloud in accordance with their privacy policies:

- **For aliyun.com users**: [Alibaba Cloud China Privacy Policy](https://terms.aliyun.com/legal-agreement/terms/suit_bu1_ali_cloud/suit_bu1_ali_cloud202107091605_49213.html)
- **For alibabacloud.com users**: [Alibaba Cloud International Terms of Service](https://www.alibabacloud.com/help/en/legal/latest/alibaba-cloud-international-website-product-terms-of-service-v-3-8-0)

**Please review Alibaba Cloud's privacy policies to understand how they collect, use, and protect your data.**

## Your Responsibilities

As a user of this Plugin, you are responsible for:

1. Ensuring you have appropriate rights to transmit any data through the Plugin
2. Complying with applicable data protection laws and regulations
3. Reviewing and accepting Alibaba Cloud's terms of service and privacy policies
4. Securely managing your Alibaba Cloud API credentials within Dify

## Data Security

- API credentials are managed by your Dify instance according to its security practices
- All communications between the Plugin and Alibaba Cloud use secure HTTPS connections
- The Plugin does not implement any local caching or logging of request/response data
